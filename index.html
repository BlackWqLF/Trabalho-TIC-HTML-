TAGS DE HTML



Tags de Estrutura

HTML

➠ A tag é a raiz da página HTML.

HEAD

➠ Contem meta-informações sobre a pagina HTML.Serve para encontrar a página mais facilmente.

TITLE

➠ Especifica um título para a página HTML ( mostrado na barra de título)

META

➠ Especificar o conjunto da descrição da página, configurações da janela de visualização.

BODY

➠ É o corpo do documento e é um recipiente paara todo o conteúdo visível,como títulos, imagens,etc..

DOCTYPE

➠ É a declaração que define se este documento é um documento HTML5.

STYLE

➠ É o atributo de estilo para estilizar elementos HTML.

LINK

➠ É o elemento que especifica as relações entre o documento atual e um recurso externo.


Tags de Texto

P

➠ Representa um parágrafo.

De H1 até H6

➠ Representam seis níveis de título ou legendas.

Exemplo:

H1 - HTML

H2 - HTML

H3 - HTML

H4 - HTML

H5 - HTML
H6 - HTML

STRONG

➠ Utilizado em conteúdos que são de “grande importância”, incluindo coisas de urgentes(como alertas).

Exemplo:

Dia está muito bonito hoje.

B

➠ Coloca texto em negrito

Exemplo:

Esta noite está muito Linda

I

➠ Parte do texto que é destacada do restante por algum motivo.

U

➠ Utilizado para sublinhar um espaço definido de um texto.

Mark

➠ Representa um trecho de destaque em um texto.

Exemplo:

Hoje acordei cedo.

BR

➠ Produz uma quebra linha em um texto(poemas ou endereço).

HR

➠ Representa um quebra temática entre elementos de nível de parágrafo.

Base

➠ Utilizada por todos os endereços relativos contidos dentro de um documento.

IMG

➠ A tag IMG é usada para incorporar uma imagem em uma página da web.

Tags de Listas

UL

➠ A tag UL apresenta uma lista não ordenada. Cada item da lista começa com a tag LI.

Exemplo:

☉ Jez

☉ Expresso

☉ Wildcard

☉ Tag

OL

➠ A tag OL apresenta uma lista ordenada. Cada item da lista começa com a tag LI.

Exemplo:

1. Real Madrid

2. Internazionale

3. Sheriff Tirspol

4. Shaktar Donetsk

LI

➠ A tag LI define uma lista.

DL, DD e DT

➠ A tag DL define uma lista de descrição, tag DT define o nome (termo) e a tag DD descreve cada termo:

Tag

- Charles Leclerc ganhou o último grand prix de Itália.

Expresso

- Max Verstappen é o atual campeão de Fórmula 1.


Tags de Tabelas

TABLE, TH, TR e TD

➠ A tag TABLE define uma tabela, a tag TH é o cabeçalho da tabela, TR define uma coluna e a tag TD define célula na tabela.

Número de Champions ganhas
Clube Nº Champions ganhas
Real Madrid 13 quase 14
AC Milan 7
Liverpool 6

TBODY

➠ Agrupa conteúdo do corpo na tabela.

TFOOT

➠ Agrupa conteúdo do rodapé na tabela.

COL

➠ Especifica as propriedades da coluna para cada coluna dentro de um elemento COLGROUP.

COLGROUP

➠ Especifica um grupo de uma ou mais colunas em uma tabela para formatação.

CAPTION

➠ Define uma legenda de tabela.


Tags de Forms

FORM

➠ O elemento FORM é usado para criar um formulário HTML para entrada do usuário.

➠ O elemento FORM é um espaço para diferentes tipos de elementos de entrada, coomo: botões, campos de seleção e caixas de seleção etc.

INPUT

➠ A tag INPUT é o elemento de formulário mais usado.

➠ A tag INPUT pode ser exibido de várias maneiras, dependendo do atributo/tag TYPE.

Exemplo:

TEXTAREA

➠ A tag TEXTAREA define um campo de entrada de várias linhas (uma área de texto).

Exemplo:

SELECT e OPTION

➠ A tag SELECT define uma lista suspensa que trabalha em junção com os elementos OPTION que definem uma opção que pode ser selecionada.




OPTGROUP

➠ A tag OPTGROUP define um grupo de opções relacionadas em uma lista suspensa.

BUTTON

➠ A tag BUTTON define um botão clicável:

Olá

LABEL

➠ A tag LABEL define um rótulo para vários elementos de formulário.

➠ A tag LABEL é útil para usuários de leitores de tela, porque o leitor de tela lê em voz alta o rótulo quando o usuário focar no elemento de entrada.

FIELDSET e LEGEND

➠ A tag FIELDSET é usado para agrupar dados relacionados em um formulário.

➠ A tag LEGEND define uma legenda para o elemento FIELDSET.


Tags de Scripting

SCRIPT

➠ A tag SCRIPT é usada para incorporar uma script do lado do cliente (JavaScript).

NOSCRIPT

➠ A tag NOSCRIPT define um conteúdo alternativo a ser exibido para usuários que desabilitaram scripts em seus navegadores ou têm um navegador que não suporta o script.
